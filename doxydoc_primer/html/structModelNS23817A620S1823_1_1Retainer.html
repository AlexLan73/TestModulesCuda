Detailed Description

template<class T>
struct ModelNS23817A620S1823::Retainer< T >

Простейший пул, предназначенный для того, чтобы не отдавать память. При работе модели с деревом станции постоянно создаются и уничтожаются объекты ячеек (Cell412_06 и прочие), и, судя по выдаче time (значения sys в десятки раз больше, чем без дерева), постоянное выделение и освобождение блоков памяти в сотни мегабайт очень дорого по времени (real, реальному времени, хотя user меняется мало). Поэтому делаем пул, который на самом деле не освобождает память, а придерживает ее для последующего переиспользования.
